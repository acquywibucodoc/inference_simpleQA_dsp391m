This folder contains the frontend files for the ChatGPT-like web UI.

To run the app:
1. Make sure you have FastAPI and Uvicorn installed (pip install fastapi uvicorn pydantic).
2. Start the backend with: python app.py
3. Open http://localhost:8000 in your browser.

The background image is set to ../img/background.jpeg. If you move the static folder, update the CSS path accordingly.
